# RegistrationApp

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Приложение для регистрации и авторизации пользователей с передачей данных между Activity.

## О проекте

Приложение демонстрирует систему регистрации и входа пользователей с использованием Activity Result API. Включает разные экраны для регистрации и авторизации.

**Основной функционал:**
- Регистрация нового пользователя
- Авторизация существующего пользователя
- Передача данных между Activity
- Выбор аватара
- Валидация учетных данных

## Технологии

* Kotlin
* ViewBinding
* Activity Result API
* Multiple Activities
* Data passing
